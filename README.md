# DFA Game

Build [deterministic finite automata (DFAs)](https://en.wikipedia.org/wiki/Deterministic_finite_automaton) using a visual editor and test them against challenges.

## Files

- `index.html` — editor and game UI
- `main.py` — FastAPI backend (`/check`, `/submit`)
- `dfa.py` — DFA logic (load YAML, validate, run, compare)
- `challenges.json` — challenge definitions
- `solutions/` — solution YAML files

Feel free to suggest challenge ideas!
